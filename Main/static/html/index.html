小目标检测

遥感影像的应用以拍摄大范围观测场景来获取信息为主要目的，图像的尺寸相较于一般图像会更加庞大。此外，遥感拍摄的场景多为复杂的室外场景，常面临曝光、阴影、背景遮挡和干扰等问题。依上，遥感图像中的小目标在整张图像中的所占面积极小，分辨率低，且常遭受背景噪声的干扰。

多尺度目标检测

面向遥感影像的目标检测，常为多类别、多目标的复杂情形检测。众多的检测类别中，不乏有上述提及的小型检测目标，如人、汽车、小船等，但同时也存在大型检测目标，如飞机、篮球场、海港、服务站等。据上，遥感场景的目标呈现巨大尺度变化，包括类内尺度变化、类间尺度变化以及不同场景下的形态变化、目标旋转变化等。
